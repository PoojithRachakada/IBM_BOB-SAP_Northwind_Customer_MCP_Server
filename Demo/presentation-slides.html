Setup & Configuration

MCP Server Configuration

{
"mcpServers": {
"northwind": {
"command": "node",
"args": [
"C:/path/to/northwind-server/build/index.js"
],
"disabled": false,
"alwaysAllow": [],
"disabledTools": []
}
}
}

📦 Installation Steps

  • Install Node.js
  • Clone MCP server repository
  • Run npm install
  • Build with npm run build
  • Configure in Bob settings

✅ Verification

  • Open Bob AI Assistant
  • Check MCP server status
  • Should show "northwind" connected
  • Test with simple query
  • Ready to demo!

OData Query Parameters

All tools support flexible OData query parameters:

🔍 Filtering

filter: "Country eq 'USA'"
filter: "UnitPrice gt 20"
filter: "City eq 'London' and
ContactTitle eq 'Owner'"

📊 Sorting

orderby: "CompanyName asc"
orderby: "UnitPrice desc"
orderby: "OrderDate desc"

📄 Pagination

top: 10 // First 10 records
skip: 20 // Skip first 20
top: 25, skip: 50 // Page 3

🎯 Field Selection

select: "CustomerID,
CompanyName,
City"

Traditional vs MCP Approach

❌ Traditional Approach

c.CompanyName,
c.ContactName,
c.City
FROM Customers c
WHERE c.City = 'London'
ORDER BY c.CompanyName;

  • Need database access
  • Manual query writing

  • Time-consuming
